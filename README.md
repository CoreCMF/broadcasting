# corecmf

## Structure

```     
src/
```

## Install

Via Composer

```bash
$ composer require corecmf/broadcasting
```

## Usage
安装完成后需要在config/app.php中注册服务提供者到providers数组：
```
CoreCMF\Broadcasting\BroadcastingServiceProvider::class,
```
##install   
```
php artisan corecmf:broadcasting:install
```
